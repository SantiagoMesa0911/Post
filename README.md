
# Proyecto de Creación de Posts

Este es un proyecto que te permite crear y compartir tus propios posts de manera sencilla y elegante. Utilizando una combinación de tecnologías poderosas, hemos creado una aplicación web que hace que la creación y visualización de contenido sea rápida y agradable.

## 🛠 Skills
- PHP
- HTML Y CSS
- Javascript 



## Tech Stack

**Client:** React, Inertia, Bootstrap

**Server:** Laravel
